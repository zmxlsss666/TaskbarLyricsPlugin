# TaskbarLyricsPlugin
一个适用于 Salt Player For Windows 的任务栏歌词插件
## 安装方法
1. 从 [Release页面](https://github.com/zmxlsss666/TaskbarLyricsPlugin/releases) 下载最新版本
2. 进入SPW→设置→创意工坊→模组管理→导入→TaskbarLyricsPlugin→启用

## 注意事项
仅适配原版任务栏，对于微软商店版未做测试，目前仅任务栏对齐方式靠左可用（偷懒）
